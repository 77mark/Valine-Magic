# Valine-Magic-BQB

这是Valine-Magic（魔改版本）的表情包库，使用jsdelivr进行CDN加速

会考虑做分类表情，但是肯定没那么快

如果你有新的表情包想要加入，你可以提出issue，或者直接提交PR让我把表情包并入此库

仓库内的那个PY脚本是我提前编写好用来写表情列表的脚本，如果你有需要可以随意取用
